### SmartSizer

SmartSizer is a Windows desktop application built with Python for efficiently resizing images. 
It supports both single and batch processing, allowing users to resize multiple images at once with ease. 
The application provides a real-time preview of output, helping users adjust dimensions before applying changes.
With a modern user interface featuring dark and light themes, along with clear progress indicators during conversion, SmartSizer delivers a smooth and user-friendly experience for handling image resizing tasks.

---

## Current Features

### Image Processing
* **Single Image Resizing:** Quickly resize individual images with precision.
* **Batch Processing:** Resize multiple images at once to save time and effort.
* **Flexible Dimensions:** Customize width and height based on your needs.

### User Experience
* **Real-Time Preview:** Instantly visualize output before applying changes.
* **Dark / Light Themes:** Switch between modern UI themes for better usability.
* **Clean Interface:** Simple and intuitive design focused on productivity.

### Performance & Feedback
* **Progress Indicators:** Track resizing progress during batch operations.
* **Efficient Processing:** Optimized handling for multiple images.
* **Error Handling:** Prevents crashes and handles invalid inputs gracefully.
